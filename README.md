# AWFISYSTEM （本科毕业设计）
基于相似度的匿名流量网站指纹识别系统

使用优化过的编辑距离算法对OpenSSH匿名流量进行识别网站类别
可部署本地抓取流量或自定义上传匿名流量包进行识别
识别率可达百分之80以上
